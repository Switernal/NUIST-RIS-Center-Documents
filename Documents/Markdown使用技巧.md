# Gitbook

## 前言

使用 Gitbook 可以轻松将 Markdown 文件转换为电子书，并部署在服务器上。本在线文档即使用Gitbook服务生成。

## 注意事项

### 1、表格

在普通 Markdown 文件中插入表格较为随意，但是在 Gitbook 中，表格必须顶格写（紧贴左侧行头，不能有空格、缩进）
