# 报告排版具体要求

## 1. 封面

> 学校 Logo 图片：

| 宽度       | 段落格式 | 图片布局   |
| :-: | :-: | :-: |
| **16**厘米 | 居中     | 上下环绕型 |

> 横向标题（问卷标题）：

| 字体                       | 字号               | 段落格式 |
| :-: | :-: | :-: |
| **华文行楷**或**华文新魏** | **45**，可按需调节 | **居中** |

> 竖向标题（调研报告四个字）：

| 字体     | 字号 | 段落格式 |
| :-: | :-: | :-: |
| 华文楷体 | 55   | 居中     |

> 右下角信息：

|        | 字体 | 字号 | 格式 | 段落格式               |
| :-: | :-: | :-: | :-: | :-: |
| 第一行 | 楷体 | 小二 | 加粗 |                        |
| 第二行 | 楷体 | 小二 | 加粗 | 相对居中（段落中设置） |
