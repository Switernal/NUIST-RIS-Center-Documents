| 宽度       | 段落格式 | 图片布局   |
| ---------- | -------- | ---------- |
| **16**厘米 | 居中     | 上下环绕型 |

| 字体     | 字号 | 段落格式 |
| -------- | ---- | -------- |
| 华文楷体 | 55   | 居中     |

|        | 字体 | 字号 | 格式 | 段落格式               |

| 第二行 | 楷体 | 小二 | 加粗 | 相对居中（段落中设置） |
